# 介绍
『无名氏服务器插件』是一个基于原版GameTest框架的插件，让原生BDS也可以拥有很多基础的功能。
目前下载量已超过3w，期待你的使用
![image](upload/202402/IMG_0952.jpeg)

## 前置环境
| 项目          | 需求                                   |
| ----------- | ------------------------------------ |
| Minecraft版本 | Minecraft基岩版1.19.50及以上               |
| 实验性选项       | 测试版API , 实验性相机功能(V2)                 |
| 服务端类型       | BDS/llse/Endstone/Realms或Minecraft游戏 |
| 服务端版本       | 与Minecraft版本相同                       |
| 日志环境        | node.js 20+                          |

*注:日志环境非必须，但你要用日志功能就必须安装node.js。*
*服务端类型支持全部拥有Minecraft基岩版原版特性的服务端，也就是说支持的服务端类型不仅仅包括上面提到的类型。*
